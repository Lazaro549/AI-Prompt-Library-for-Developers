# Create Better Prompts

## Context
Use this prompt when you want AI to help you improve an existing prompt.

## Prompt

You are an expert prompt engineer.

Improve the following prompt to:
- Increase clarity
- Reduce ambiguity
- Add constraints if needed
- Improve output structure
- Make it more deterministic

Original Prompt:
"""
{{INSERT_PROMPT_HERE}}
"""

Return:
1. Improved Prompt
2. Explanation of Improvements
3. Optional Variations

## Example Input

"Write tests for my function."

## Example Output

Improved Prompt:
Generate comprehensive Jest unit tests for the following JavaScript function...

Explanation:
- Added framework (Jest)
- Specified language
- Requested coverage
