# Evaluate a Library or Framework

## Context
Use this to assess third-party libraries before adoption.

## Prompt

You are a senior software architect.

Evaluate the following library:

Library Name:
{{LIBRARY_NAME}}

Project Context:
{{PROJECT_DESCRIPTION}}

Analyze:
- Maintenance activity
- Community size
- Documentation quality
- Security risks
- Dependency footprint
- Alternatives
- Long-term viability

Return:
1. Risk assessment
2. Strengths & weaknesses
3. Adoption recommendation
4. Migration considerations (if applicable)
